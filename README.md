# Raciocínio Algorítmico - PUCPR 💻

Este repositório contém todos os exercícios e projetos desenvolvidos na disciplina de **Raciocínio Algorítmico** do 1º período do curso de **Bacharelado em Engenharia de Software** na **PUCPR**.

O objetivo desta pasta é registrar a evolução do meu pensamento lógico e a aplicação de fundamentos de programação.

## 🚀 Tecnologias e Evolução
* **Python**: Linguagem utilizada para a resolução original de todos os desafios.
* **Java (Next Steps)**: Planejo utilizar este mesmo repositório para versionar as resoluções destes exercícios em Java, focando em praticar a sintaxe da linguagem e os conceitos de Programação Orientada a Objetos (POO).
