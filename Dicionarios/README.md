# 📖 Lista de Exercícios – Dicionários

1. **Informações de Pessoa**: Dicionário com nome, idade e cidade; busca por chave.
2. **Preço de Produtos**: Atualização de preços via input do usuário.
3. **Cadastro Simples**: Dicionário vazio que armazena nome e idade informados.
4. **Função dict()**: Construir dicionário a partir de três pares de dados.
5. **Método clear()**: Opção para apagar todos os dados do dicionário.
6. **Cópia de Dados**: Comparação entre dicionário original e cópia alterada.
7. **fromkeys()**: Criar dicionário de nomes com valor padrão 0.
8. **Busca de Notas**: Uso do método `get()` para buscar notas de alunos.
9. **Visualização**: Exibir separadamente chaves, valores e itens.
10. **Manipulação**: Uso de `pop()`, `popitem()` e `update()`.

### 11. Sistema de Gerenciamento de Usuários
Desenvolvimento de um sistema completo com menu interativo permitindo:
- Exibir usuários (Nomes, Idades ou Pares).
- Buscar usuário por nome (`get`).
- Adicionar e Atualizar usuários.
- Remover usuário específico ou o último inserido.
- Criar cópia para comparação.
- Inicialização em massa via `fromkeys`.
- Limpeza total do sistema.
