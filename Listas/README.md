### Lista de Exercícios: Listas e Vetores

1. **Criando uma Lista**: Crie uma lista com os números de 1 a 5 e imprima-a na tela.

2. **Acesso por Índice**: Dada a lista `cores = ['vermelho', 'azul', 'verde', 'amarelo']`, acesse e imprima a segunda cor da lista.

3. **Adicionando Elemento**: Adicione o número `10` ao final da lista `numeros = [1, 2, 3]`.

4. **Removendo Elemento**: Remova a palavra `'banana'` da lista `frutas = ['maçã', 'banana', 'laranja']`.

5. **Tamanho da Lista**: Encontre e imprima a quantidade de elementos da lista `itens = [10, 20, 30, 40, 50]`.

6. **Verificação de Pertencimento**: Verifique se o número `7` está presente na lista `valores = [1, 3, 5, 7, 9]` e imprima o resultado booleano.

7. **Concatenação de Listas**: Concatene as listas `lista1 = [1, 2]` e `lista2 = [3, 4]` em uma terceira lista.

8. **Inversão de Lista**: Inverta a ordem dos elementos da lista `letras = ['a', 'b', 'c', 'd']`.

9. **Contagem de Ocorrências**: Conte quantas vezes o número `2` aparece na lista `numeros = [1, 2, 2, 3, 2, 4]`.

10. **Soma dos Elementos**: Calcule e imprima a soma de todos os elementos da lista `precos = [10.5, 20.0, 15.5]`.

11. **Remoção de Duplicatas**: Escreva um programa que remova as duplicatas de uma lista, garantindo que a ordem original dos primeiros elementos encontrados seja mantida.

12. **Máximo e Mínimo Manual**: Encontre o maior e o menor número em uma lista de inteiros **sem usar** as funções nativas `max()` e `min()`.

13. **List Comprehension — Quadrados**: Use list comprehension para criar uma lista contendo os quadrados dos números de 1 a 10.

14. **Filtrando Ímpares**: Dada uma lista mista de números, crie uma nova lista contendo apenas os números ímpares da lista original.

15. **Rotação de Lista**: Escreva um programa que rotacione os elementos de uma lista para a direita em `n` posições.
    > **Exemplo:** a lista `[1, 2, 3, 4, 5]` com `n=2` resulta em `[4, 5, 1, 2, 3]`.

16. **Interseção sem `set`**: Dadas duas listas, encontre a interseção entre elas (os elementos presentes em ambas) **sem usar** conversão para conjuntos (`set`).

17. **Achatamento de Lista**: "Achate" (*flatten*) uma lista de listas em uma única lista unidimensional.
    > **Exemplo:** `[[1, 2], [3, 4]]` vira `[1, 2, 3, 4]`.

18. **Merge Sort**: Implemente o algoritmo de ordenação *Merge Sort* (Ordenação por Mistura) para ordenar uma lista desordenada de números em ordem crescente.

19. **Algoritmo de Kadane**: Dada uma lista de números inteiros (positivos e negativos), encontre a sublista contígua com a maior soma e retorne o valor dessa soma.

20. **Permutações Recursivas**: Escreva uma função recursiva que receba uma lista de números distintos e retorne todas as permutações possíveis de seus elementos.
