# 📋 Lista de Exercícios – Listas e Vetores

1. Criar e imprimir lista de 1 a 5.
2. Acessar a segunda cor de uma lista de cores.
3. Adicionar o número 10 ao final de uma lista.
4. Remover "banana" de uma lista de frutas.
5. Imprimir a quantidade de elementos de uma lista.
6. Verificar se o número 7 está presente na lista (retorno booleano).
7. Concatenar duas listas em uma terceira.
8. Inverter a ordem dos elementos de uma lista.
9. Contar ocorrências do número 2 em uma lista.
10. Calcular a soma de todos os elementos (preços).
11. Remover duplicatas mantendo a ordem original.
12. Encontrar maior e menor número sem usar `max()` ou `min()`.
13. List Comprehension para criar lista de quadrados de 1 a 10.
14. Criar nova lista apenas com números ímpares de uma lista mista.
15. Rotacionar elementos de uma lista para a direita em *n* posições.
16. Encontrar a interseção entre duas listas (sem usar `set`).
17. Achatar (flatten) uma matriz bidimensional em uma lista unidimensional.
18. Implementação do algoritmo **Merge Sort**.
19. Encontrar a sublista contígua com a maior soma (**Algoritmo de Kadane**).
20. Função recursiva para retornar todas as permutações de uma lista.
