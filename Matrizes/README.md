# 🔢 Lista de Exercícios – Matrizes

### Parte 1: Lógica Geral
1. **Soma Total**: Soma de todos os elementos de uma matriz 3x3.
2. **Identidade**: Gerar matriz identidade de ordem *n*.
3. **Busca Simples**: Verificar se um número está em uma matriz 4x4.
4. **Troca de Valores**: Trocar a primeira linha com a segunda em uma matriz 2x2.
5. **Multiplicação por Escalar**: Multiplicar cada elemento por um número real.
6. **Contagem de Pares**: Contar números pares em uma matriz 3x4.
7. **Maior Elemento**: Encontrar o maior valor em uma matriz aleatória.
8. **Média por Linha**: Média aritmética de cada linha de uma matriz 3x3.
9. **Soma da Diagonal Principal**: Soma dos elementos da diagonal em matriz 4x4.
10. **Matriz Transposta**: Gerar a transposta (N x M) de uma matriz (M x N).
11. **Soma de Colunas**: Gerar lista com a soma de cada coluna de uma matriz 3x3.
12. **Verificação de Simetria**: Verificar se a matriz é igual à sua transposta.
13. **Diagonal Secundária**: Exibir elementos da diagonal secundária de matriz 5x5.
14. **Multiplicação de Matrizes**: Implementação da multiplicação A x B.
15. **Rotação 90°**: Rotacionar matriz quadrada no sentido horário sem bibliotecas.

### Parte 2: Fixação com NumPy
1. **Monitoramento de Chuva**: Soma de matrizes (Manhã + Tarde).
2. **Controle de Estoque**: Subtração de matrizes (Inicial - Vendidos).
3. **Sistema de Lanchonete**: Multiplicação de matrizes (Ingredientes x Pedidos).
4. **RH**: Reajuste de 10% nos salários (Multiplicação por escalar).
5. **Limpeza de Sistema**: Zerar todos os valores da matriz.
6. **Sensores Ativos**: Alterar todos os valores da matriz para 1.
7. **Correção de Erro**: Substituir valores em posições específicas (L1,C2 e L3,C5).
